Votre site web.

Créez et publiez rapidement et facilement votre propre site web en utilisant iWeb, Pages et de nombreuses autres applications disponibles
pour Mac OS X.

Il est extrêmement simple de créer et de publier votre propre site web à partir de votre Mac. Lorsque votre site est prêt, il est tout aussi simple de le publier.

Ouvrez Préférences Système et cliquez sur Partage, puis sélectionnez Partage web.

Vous avez terminé. Votre site est désormais disponible sur votre réseau privé, chez vous ou au bureau.

Si vous êtes connecté à Internet, votre site web peut également être mis à la disposition de vos amis, où qu’ils se trouvent. Il vous suffit de leur envoyer l’adresse indiquée dans les préférences Partage.

La puissance d’Apache

Le partage web est conçu autour du serveur web Apache, une technologie standard fournie avec Mac OS X. Pour plus de renseignements sur le serveur web Apache, consultez le manuel Apache.
